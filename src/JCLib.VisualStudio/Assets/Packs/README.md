Depuis la version `1.3.12`, `Ctrl+Alt+P` ouvre une fenêtre Find Symbol vide avec focus immédiat sur la saisie.
Depuis la version `1.3.11`, l’interface WPF distingue visuellement les niveaux de hiérarchie par des icônes, couleurs et badges configurables. Les catalogues JSON distribués restent identiques à la version 1.3.10.

Depuis la version `1.3.10`, les catalogues autonomes `default_pack.json`, `build_pack.json`, `cvi_pack.json`, `system_scripting_pack.json`, `web_language_pack.json`, `csharp_language_pack.json`, `c_language_pack.json`, `cpp_language_pack.json` et `windows_api_device_pack.json` sont distribués avec les sources Visual Studio. Le navigateur Visual Studio reste un consommateur/importeur de packs : la génération des packs intégrés reste réservée à l’extension VS Code. Le moteur de preview prend en charge les sélecteurs structurés, les paramètres conditionnels, `insertValueMap`, les champs multi-lignes, l’ordre canonique des choix multiples et la suppression des attributs HTML / CSHTML cités laissés vides.

Catalogues synchronisés avec VS Code : `web_language_pack.json` 2.0.0, packs C/C++/Win32 DLL modernisés et fallback courant.
