Depuis la version `1.3.9`, les catalogues autonomes `default_pack.json`, `build_pack.json`, `cvi_pack.json`, `system_scripting_pack.json`, `web_language_pack.json` et `csharp_language_pack.json` sont distribués avec les sources Visual Studio. Le navigateur Visual Studio reste un consommateur/importeur de packs : la génération des packs intégrés reste réservée à l'extension VS Code. Le moteur de preview prend en charge les sélecteurs structurés, les paramètres conditionnels et la suppression des attributs HTML / CSHTML cités laissés vides.
