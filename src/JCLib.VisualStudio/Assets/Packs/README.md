Depuis la version `1.3.7`, les catalogues autonomes `default_pack.json`, `build_pack.json`, `cvi_pack.json` et `system_scripting_pack.json` sont distribués avec les sources Visual Studio. Le navigateur Visual Studio reste un consommateur/importeur de packs : la génération des packs intégrés reste réservée à l'extension VS Code. Le bouton `Exécuter dans une console` ouvre une console externe adaptée pour les cartes `command` et les recettes `Scripting / System`.
