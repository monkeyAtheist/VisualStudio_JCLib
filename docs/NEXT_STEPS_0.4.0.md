# Étapes suivantes après JC Lib Visual Studio 0.4.0

La prochaine itération pourra ajouter les packs globaux et liés à la solution :

- découverte des fichiers JSON utilisateur ;
- priorité des packs de solution sur les packs globaux ;
- bouton d'import d'un pack ;
- rechargement consolidé de plusieurs packs ;
- affichage de la provenance de chaque pack.

Le Visual Pack Editor complet et l'intégration WebView2 peuvent rester une phase ultérieure, après validation du chargement multi-pack natif.
