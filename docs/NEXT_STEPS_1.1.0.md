# Prochaine passe proposée après 1.1.0

La base fonctionnelle est suffisante pour une utilisation quotidienne. Les améliorations suivantes peuvent être traitées séparément :

- raccourcis configurables depuis une page d'options JC Lib ;
- tri manuel des favoris ;
- export et import des préférences utilisateur ;
- intégration WebView2 optionnelle pour rapprocher visuellement l'interface de la version VS Code ;
- packaging de distribution et procédure d'installation hors instance expérimentale ;
- tests automatisés Windows avec MSBuild et VSSDK installés.
