# JC Lib Visual Studio 0.8.0 — création de packs et éditeur hiérarchique

Cette version étend le Visual Pack Editor WPF introduit en 0.7.0.

## Ajouts principaux

- création d'un pack de démarrage global ou lié à la solution ;
- duplication d'un pack sélectionné, y compris du catalogue embarqué ;
- création, renommage et suppression d'environnements, bibliothèques, catégories et groupes ;
- suppression récursive d'un sous-arbre après confirmation ;
- édition graphique des paramètres structurés ;
- validation des doublons hiérarchiques et des paramètres ;
- autorisation des packs externes vides afin de permettre leur construction progressive dans l'éditeur.

## Compatibilité

Le navigateur multi-pack, la politique de priorité, la recherche, la prévisualisation et l'insertion dans l'éditeur actif restent inchangés.
