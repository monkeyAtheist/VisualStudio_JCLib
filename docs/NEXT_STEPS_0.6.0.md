# Étapes suivantes après JC Lib Visual Studio 0.6.0

La prochaine itération pourra ajouter la création et l'édition de packs :

- création d'un pack vide global ou lié à la solution ;
- duplication d'un pack existant ;
- ajout, édition et suppression d'éléments ;
- validation des noms et des chemins logiques ;
- préparation du portage du Visual Pack Editor.

Le Visual Pack Editor complet pourra ensuite être porté soit en WPF natif, soit via une Tool Window hybride WebView2 réutilisant une partie de l'interface HTML existante.
