# Étapes suivantes après JC Lib Visual Studio 0.8.0

La version 0.8.0 couvre la création de packs, la duplication, l'édition hiérarchique et les paramètres structurés.

La prochaine passe pourra ajouter :

- déplacement d'éléments et de groupes vers un nouveau parent ;
- réordonnancement par boutons monter / descendre ;
- duplication d'un élément ou d'un sous-arbre ;
- sélection multiple ;
- import et export partiels d'une bibliothèque, catégorie ou groupe ;
- presets d'éléments et modèles de création ;
- aperçu du snippet généré directement dans le Visual Pack Editor.

Une passe ultérieure pourra comparer l'intérêt d'un éditeur WPF natif complet avec une intégration WebView2 réutilisant une partie de l'interface HTML de l'extension VS Code.
