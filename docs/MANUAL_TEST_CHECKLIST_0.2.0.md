# Checklist manuelle — JC Lib Visual Studio 0.2.0

1. Compiler en `Debug | Any CPU`.
2. Lancer avec `F5`.
3. Dans l'instance `EXP`, ouvrir `Affichage > Autres fenêtres > JC Lib`.
4. Vérifier que le résumé affiche environ `2 042 éléments`.
5. Déplier quelques niveaux de l'arborescence.
6. Rechercher `QMessageBox information` puis sélectionner un résultat.
7. Vérifier le prototype et le snippet.
8. Cliquer sur `Copier le snippet`, puis coller dans un éditeur texte.
9. Effacer la recherche et naviguer dans `Lua`.
10. Cliquer sur `Recharger`.
