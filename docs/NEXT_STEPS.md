# Étapes suivantes après JC Lib Visual Studio 1.0.0

La version 1.0.0 couvre le noyau fonctionnel du Visual Pack Editor : gestion multi-pack, édition hiérarchique, paramètres structurés, opérations groupées, déplacement et duplication de sous-arbres, import et export partiels.

Pistes pour une prochaine passe :

- glisser-déposer visuel dans l'arborescence ;
- annuler / rétablir les opérations ;
- export partiel d'une sélection d'éléments ;
- édition assistée des presets et options ;
- comparaison de deux packs et fusion contrôlée ;
- raccourcis clavier pour les opérations fréquentes.
