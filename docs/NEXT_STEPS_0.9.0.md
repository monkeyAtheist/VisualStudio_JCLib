# Étapes suivantes après JC Lib Visual Studio 0.9.0

La version 0.9.0 couvre la sélection multiple, la duplication, la suppression groupée, le changement de parent des éléments et le réordonnancement.

La prochaine passe pourra ajouter :

- déplacement complet d'une catégorie, d'un groupe ou d'un sous-arbre vers un nouveau parent ;
- duplication d'un groupe avec son contenu ;
- import et export partiels d'une bibliothèque, catégorie ou groupe ;
- aperçu du snippet généré directement dans le Visual Pack Editor ;
- filtres dans l'arborescence d'édition ;
- raccourcis clavier et commandes contextuelles.

Une passe ultérieure pourra comparer l'intérêt d'un éditeur WPF natif complet avec une intégration WebView2 réutilisant une partie de l'interface HTML de l'extension VS Code.
