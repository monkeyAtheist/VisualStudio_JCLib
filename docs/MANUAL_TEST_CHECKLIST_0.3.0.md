# Checklist manuelle — JC Lib Visual Studio 0.3.0

1. Compiler le projet en `Debug | Any CPU`.
2. Lancer l'instance expérimentale avec `F5`.
3. Ouvrir `Affichage > Autres fenêtres > JC Lib`.
4. Ouvrir un fichier source modifiable, par exemple `test.cpp`.
5. Rechercher `QTimer periodic timeout` ou `QMessageBox information`.
6. Sélectionner une entrée puis cliquer sur `Insérer dans l'éditeur`.
7. Vérifier que le snippet apparaît à la position du curseur.
8. Placer le curseur dans un bloc indenté et répéter l'insertion : les lignes suivantes doivent conserver l'indentation de base.
9. Sélectionner une portion de texte et insérer un snippet : la sélection doit être remplacée.
10. Fermer tous les éditeurs texte et cliquer sur l'insertion : un message doit indiquer qu'aucun éditeur texte n'est actif.
11. Vérifier que `Copier le snippet` fonctionne toujours.
