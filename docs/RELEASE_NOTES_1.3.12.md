# Release notes 1.3.12 — Find Symbol global

- Ajoute une fenêtre WPF autonome `Find Symbol`.
- Ajoute le raccourci global configurable `Ctrl+Alt+P`.
- Place immédiatement le focus dans la zone de saisie.
- N'affiche aucun résultat et aucune preview tant que la requête est vide.
- Conserve le champ de recherche intégré au navigateur principal.
- Synchronise le catalogue Web autonome en version 2.0.0.
- La génération des packs intégrés reste réservée à l'extension VS Code.
