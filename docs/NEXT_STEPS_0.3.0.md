# Étape suivante proposée — 0.3.0

Ajouter l'insertion dans l'éditeur Visual Studio actif :

- récupérer le document et la sélection courante ;
- remplacer la sélection ou insérer au curseur ;
- conserver l'indentation de la ligne active ;
- ajouter un bouton `Insérer dans l'éditeur` ;
- conserver le bouton `Copier` comme solution de secours.
