# Étapes suivantes après JC Lib Visual Studio 0.7.0

Le Visual Pack Editor minimal valide la chaîne complète : ouverture, modification JSON, validation, sauvegarde et rechargement.

La prochaine passe pourra ajouter :

- création d'un pack vide global ou lié à la solution ;
- duplication d'un pack existant ;
- ajout, suppression et renommage des environnements, bibliothèques, catégories et groupes ;
- édition graphique des paramètres structurés ;
- déplacement d'éléments entre groupes ;
- réordonnancement ;
- import et export partiels ;
- sélection multiple.

Une passe ultérieure pourra comparer l'intérêt d'un éditeur WPF natif complet avec une intégration WebView2 réutilisant une partie de l'interface HTML de l'extension VS Code.
